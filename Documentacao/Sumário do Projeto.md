## 📄 Descrição de cada arquivo

### **T2.py** ⭐ EXECUTÁVEL PRINCIPAL

**O que é:** Programa principal do projeto, interface com usuário

**Funções principais:**
- `lerArquivo()` - Lê arquivo de preferências
- `imprimirPreferencias()` - Exibe preferências formatadas
- `executarModoAutomatico()` - Executa AG sem pausa
- `executarModoPassoAPasso()` - Executa AG com pausa entre gerações
- `main()` - Gerencia argumentos de linha de comando

**Como usar:**
```bash
python T2.py exemplo1.txt
python T2.py exemplo1.txt --modo passo
python T2.py exemplo1.txt --pop 100 --ger 1000
```

**Localização:** `/mnt/user-data/outputs/T2.py`

---

### **Genetico.py**

**O que é:** Implementação do Algoritmo Genético

**Funções principais:**
- `geraAptidao()` - Calcula qualidade de uma solução
- `popula()` - Cria população inicial aleatória
- `ordena()` - Ordena população por aptidão
- `getMediaAptidao()` - Calcula média de aptidão
- `selecaoTorneio()` - Seleção por torneio
- `selecaoRodaDaRoleta()` - Seleção por roda da roleta
- `crossover()` - Order Crossover (OX)
- `mutacao()` - Mutação por inversão
- `novaGeracao()` - Cria próxima geração
- `AG()` - Loop principal do algoritmo
- `_log()` - Exibe progresso

**Localização:** `/mnt/user-data/outputs/Genetico.py`

---

### **exemplo1.txt**

**O que é:** Arquivo de entrada com exemplo de 4 alunos

**Formato:**
```
4                    ← Número de duplas
1 1 2 4 3           ← Preferências de A1 (B1, B2, B4, B3)
2 2 3 1 4           ← Preferências de A2
3 4 1 3 2           ← Preferências de A3
4 3 1 2 4           ← Preferências de A4
1 3 2 4 1           ← Preferências de B1 (A3, A2, A4, A1)
2 3 2 4 1           ← Preferências de B2
3 2 4 3 1           ← Preferências de B3
4 1 3 4 2           ← Preferências de B4
```

**Como usar:**
```bash
python T2.py exemplo1.txt
```

**Localização:** `/mnt/user-data/outputs/exemplo1.txt`
